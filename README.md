# Usage
<!-- usage -->
```sh-session
$ npm install -g eureka-register
$ eureka-register COMMAND
running command...
$ eureka-register (-v|--version|version)
eureka-register/0.0.0 linux-x64 node-v12.16.1
$ eureka-register --help [COMMAND]
USAGE
  $ eureka-register COMMAND
...
```
